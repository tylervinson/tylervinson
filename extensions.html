Visual Editing for Data-Driven Sitecore Widgets

Editing data-driven widgets in Sitecore traditionally requires navigating source code to manually update inline values, data attributes, and HTML structure. Content editors shouldn't need to touch markup to change a stat, a plan price, or a card label.

This Chrome extension solves that. It opens as a persistent sidebar inside the Content Editor, reads field values directly via the Sitecore SSC API, and presents a clean grouped form. Hit Apply Changes and the extension silently automates the HTML editor — opening it, writing the updated markup, and closing it — no manual WYSIWYG interaction required.

Widget support is driven entirely by a central widget-registry.json hosted on the server. Adding or updating a widget requires only editing that file — no changes to the extension itself.

Visual HTML Theme Generator for Formstack Documents

Building custom HTML themes for Formstack Documents typically requires manual HTML editing, CSS knowledge, and tedious copy-paste workflows between form builders and document templates. Content creators shouldn't need to write code to customize how their forms look when converted to documents.

This Chrome extension eliminates that friction. It scans the current page for the Formstack Initial Template, extracts the underlying form markup, and generates a fully functional, customizable HTML template in seconds. The extension also produces JSON test data specific to your form, enabling you to preview exactly how your document will render before publishing.

Everything happens in the browser—no server calls, no authentication headaches. Copy the HTML, paste it into your document's Advanced HTML Editor, test with one click, and you're done.
