Installation

1

Download the Extension

Click the "Download Extension" button above to get the latest version of the Formstack Documents HTML Theme Generator.

2

Open Chrome Extensions

Navigate to chrome://extensions in your browser or access it through the Chrome menu under More Tools > Extensions.

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner of the extensions page.

4

Install the Extension

Drag and drop the downloaded file onto the extensions page, or click "Load unpacked" and select the extension folder.

5

Confirm Installation

The extension icon should now appear in your Chrome toolbar. You're ready to start creating custom themes.

How to Use

1

Open Your Document

Navigate to your Formstack Documents dashboard and open the document you want to customize.

2

Launch the Theme Generator

Click the extension icon in your Chrome toolbar to activate the HTML Theme Generator interface.

3

Customize Your Theme

Use the visual editor to modify colors, fonts, spacing, and layout options for your document theme.

4

Preview Your Changes

Review your theme in real-time with the preview panel to ensure it meets your requirements.

5

Export and Apply

Once satisfied, export the generated HTML theme code and apply it to your Formstack document.

6

Test Your Theme

Generate a test document to verify your theme renders correctly with actual data.

7

Save and Share

Save your custom theme for future use and share it with your team members if needed.
