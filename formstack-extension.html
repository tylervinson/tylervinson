Installation

1

Download the Extension

Click the "Download Extension" button above to get the latest version of the Formstack Documents HTML Theme Generator.

2

Open Chrome Extensions

Navigate to chrome://extensions in your browser or access it through the Chrome menu under More Tools > Extensions.

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner of the extensions page.

4

Install the Extension

Click "Load unpacked" and select the unzipped extension folder.

5

Pin Extension

Once pinned the extension icon should appear in your Chrome toolbar.

How to Use

1

Retrieve the Form URL

In Formstack, navigate to the form and copy the Formstack form URL.

2

Create the Initial Template

In Formstack Documents, open the document and go to the Document Builder tab. From the input dropdown, select “Web,” paste the copied form URL, and click Import Form. Save the document.

3

Generate the HTML Template

Open the browser extension and click “Scan Current Page.” The extension will locate the Initial Template and the associated markup. Click “Snipped” to load the markup into the conversion box, then click “Generate Template.” This produces a fully functional HTML template with all fields and labels, along with JSON form data for testing, specific to the form.

4

Implement the HTML Template

Copy the generated HTML from the extension. In Formstack Documents, open the Advanced HTML Editor, replace the existing HTML with the copied content, and save the document.

5

Test the HTML Template

Copy the JSON data from the extension. In Formstack Documents, open the Test tab, scroll to the bottom, paste the JSON into the final input field, and click Test Document.
