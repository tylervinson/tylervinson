Phase 1

Build Your Formstack Form

Create your form in Formstack using the standard form builder. Add all fields, sections, conditional logic, and styling you need. Your form design is complete when it collects the exact data you want and displays the fields in the order and grouping you prefer.

No special configuration is required at this stage. Use Formstack's native features normally—field types, required fields, conditionals, and validation all work out of the box.

  • Design your form: Use Formstack's form builder interface
  • Add all field types: Text, email, phone, dropdown, checkbox, file upload, etc.
  • Set up logic: Conditional field visibility and required field rules
  • Test the form: Verify all fields and conditionals work correctly

Phase 2

Create the Formstack Document

In Formstack Documents, create a new document and use the Document Builder to import your form as the data source. Select "Web" from the input dropdown, paste your form URL, and click "Import Form." This creates the Initial Template—a rendered preview of how your form data will appear in the document.

The Initial Template serves as both the visual reference and the data source for the extension. Save your document at this point—you'll return here in Phase 4 to paste the generated HTML.

  • Open Formstack Documents: Create a new document
  • Go to Document Builder: Access the template creation interface
  • Import the form: Select "Web," paste your Formstack form URL
  • Create Initial Template: Click "Import Form" and save
Document Builder Screenshot
Phase 3

Scan & Generate HTML

With your document open in the Document Builder, click the extension icon in your browser toolbar. The sidebar opens with a "Scan Current Page" button. Click it to detect the Initial Template on the page.

  • Open your Formstack Document in Document Builder
  • Click the extension icon to open the sidebar
  • Click "Scan Current Page"
  • The extension analyzes the Initial Template
  • HTML and JSON are generated automatically
Phase 4

Customize & Deploy HTML

Copy the generated HTML from the extension and paste it into your document's Advanced HTML Editor. The HTML is fully functional and ready to use, but you can customize CSS, adjust styling, or modify the markup to match your brand.

  • Copy HTML from the extension
  • Open Advanced HTML Editor in your document
  • Paste the HTML (replace existing content)
  • Customize CSS and styling as needed
  • Save your document
Phase 5

Test & Publish

Use the test data generated by the extension to verify your document renders correctly. Copy the JSON from the extension, paste it into the Test tab, and click "Test Document." Once verified, your document is ready to use with real Formstack form submissions.

  • Copy JSON test data from extension
  • Open the Test tab in your document
  • Paste JSON into the final input field
  • Click "Test Document" and review
  • Publish your document
