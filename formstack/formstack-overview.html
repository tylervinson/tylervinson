OVERVIEW

Visual HTML Theme Generator for Formstack Documents

Building custom HTML themes for Formstack Documents typically requires manual HTML editing, CSS knowledge, and tedious copy-paste workflows between form builders and document templates. Content creators shouldn't need to write code to customize how their forms look when converted to documents.

This Chrome extension eliminates that friction. It scans the current page for the Formstack Initial Template, extracts the underlying form markup, and generates a fully functional, customizable HTML template in seconds. The extension also produces JSON test data specific to your form, enabling you to preview exactly how your document will render before publishing.

Everything happens in the browser—no server calls, no authentication headaches. Copy the HTML, paste it into your document's Advanced HTML Editor, test with one click, and you're done.

HOW IT WORKS

Scan. Generate. Deploy.

The extension works by analyzing the DOM of the Initial Template directly in Formstack Documents. When you click "Scan Current Page," it detects the template container, extracts all form fields and their structure, and generates production-ready HTML with proper semantic markup, class names, and data attributes for styling and targeting.

Unlike manual template creation, which is error-prone and requires recreation every time the form changes, this extension automates the process. When your form fields change, you rescan and regenerate in seconds. The generated HTML includes all labels, field types, and conditional logic markers so your CSS and JavaScript have complete context.

The JSON output includes sample data for every field, allowing you to test your document instantly in the Test tab without manually filling in data.

Core Capabilities

One-Click Scanning

The extension detects the Initial Template directly from the page DOM. No manual selection of forms or fields required—click "Scan Current Page" and it finds everything automatically.

Complete HTML Generation

Produces semantically correct HTML with proper grouping, labels, field types, and data attributes. The output is immediately usable in the Advanced HTML Editor.

Automatic JSON Test Data

Generates realistic test data for every field in your form. Use it instantly in the Test tab to preview your document without manual data entry.

Copy & Paste Ready

HTML and JSON are displayed in easy-to-copy text areas. Paste directly into Formstack Documents with a single click—no downloads or file management needed.

Form-Specific Markup

Generated HTML reflects your exact form structure. Field order, grouping, conditional logic markers, and data attributes are all form-specific.

No Authentication Required

The extension works entirely in the browser using the DOM of the page you're viewing. No API calls, no login tokens, no server communication needed.

Architecture & Workflow

Initial Template

The rendered form displayed in Formstack Documents' Document Builder. Contains the HTML structure, all form fields, labels, and conditional logic that makes up your form.

Content Script

Runs on the Formstack Documents page. Uses DOM traversal to locate and extract the Initial Template markup, analyze field structure, and gather all necessary data.

Generator Engine

Processes raw DOM data into clean, semantically structured HTML. Applies CSS classes, data attributes, proper grouping, and readable formatting for easy styling and customization.

Popup Interface

User-facing extension interface. Displays the "Scan Current Page" button, the generated HTML and JSON in editable text areas, and copy buttons for instant clipboard access.

Test Data Generator

Analyzes form field types and creates realistic, representative test data for each field. Produces valid JSON that matches your form's expected data structure exactly.

Browser DOM API

All operations use native browser APIs: document.querySelector, textContent, getAttribute. No external libraries or dependencies required.
