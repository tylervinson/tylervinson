How it works Paste your widget HTML — no pre-annotation needed. The generator detects three structures automatically: items (repeating elements with item-* classes, e.g. item-name, item-price), grid (cells with data-plan and data-field attributes for comparison tables), and composite (both together). It annotates the HTML, infers field types, and generates a complete Chrome extension ZIP with a grouped editor that matches how content authors think about the widget. Already-annotated HTML passes through as-is.
