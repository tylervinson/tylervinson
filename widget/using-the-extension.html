Using the Extension

How to edit widget content in the Sitecore Content Editor

Extension Overview Widget Workflow Technical Specs Using the Extension
Download Extension
1

Open the Content Editor

Navigate to Sitecore's Content Editor and find the component you want to edit in the content tree on the left.

2

Select the Component

Click the component item in the tree. The Quick Info panel on the right will show the item ID, name, and path — the extension reads this automatically.

3

Open the Extension

Click the extension icon in your Chrome toolbar to open the sidebar panel. It will detect the current component and populate the edit form immediately.

4

Edit the Fields

Update any field values in the form. Repeating items like plan cards or stat blocks are grouped individually. Standalone fields like headings appear at the top. When finished click Apply Changes. The extension silently opens the HTML editor, writes the updated markup, and closes it — no manual WYSIWYG interaction needed.

5

Save in Sitecore

Click Save in the Content Editor toolbar. Your changes are now live in Sitecore.
