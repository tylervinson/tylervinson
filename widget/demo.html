What This Demonstrates

The key behaviours on display

Live label updates

The group headers in the sidebar update as you type — this is driven by the groupLabelField in the registry definition, which tells the sidebar which field's value to use as the group heading.

Field definition drives the form

The sidebar form is entirely generated from the registry. The extension has no knowledge of "Animated Cards" — it only knows "fields with these selectors". Change the registry and the form changes with it.

Apply writes to the HTML source

In the real extension, Apply triggers a silent HTML editor automation cycle: open textarea, write updated markup, close. The widget preview here simulates the DOM update that would result on the live Sitecore page.
