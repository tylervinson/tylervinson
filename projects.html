Responsive Layout Visualizer

This tool shows graphic designers how websites rearrange content for different screen sizes. It compares two layouts side-by-side: one where content stays together properly, and one where things break apart awkwardly.

Drag the edge or click device icons to see how each layout responds. The correct version smoothly stacks from three columns to one, while the incorrect version creates gaps and misalignments. Hover over elements to see the invisible structure that makes responsive design work.

Formstack Documents HTML Template Generator

Manually converting Formstack form submissions into email templates for our Documents workflow was time-consuming and error-prone. The process required submitting test forms, extracting field labels and variable names from HTML snippets, wrapping them in Formstack's conditional syntax, and building matching test JSON with realistic data.

This extension automates the entire process by scanning the document builder, detecting all fields, and generating both a complete email template with proper {if $Variable != ''} conditionals and {$Variable} merge tags (including |insert_image modifiers for signatures), plus test JSON with sample data. The process now takes seconds instead of hours.

Time Savings: Manual vs. Automated Process

Manual Template Creation

  • Data Extraction: Submit test form, extract and match variable names to labels
  • Template Development: Write HTML structure, add conditionals, insert merge tags, configure image modifiers
  • Test Data Creation: Build JSON structure, generate realistic values, create base64 signature images
  • Integration Configuration: Map all fields in settings, configure document merge
  • Refinement: Fix formatting issues, correct typos, re-test after changes
4-12 hours

Automated with Extension

  • Generation: Click scan, extension auto-detects fields and creates template + test JSON
  • Integration: Paste into Formstack Documents
  • Testing & QC: Visual review, validate output, test all fields
  • Refinement: Minor adjustments if needed
20-30 minutes

GA4 Analytics Viewer

Debugging GA4 and GTM implementations required constantly switching between browser tabs and network monitoring tools. This side panel extension provides persistent visibility into analytics tracking without interrupting the debugging workflow.

The extension captures analytics requests in real-time, displaying tracking IDs, UTM parameters, cookies, and network traffic to Google's servers. It includes auto-refresh intervals, JSON export functionality, and one-click cookie clearing for testing fresh sessions. All data processing happens locally within the browser.

Prosano Health Savings Calculator (Proof of Concept)

Marketing leadership needed a tool to demonstrate to brokers potential ways we could increase lead generation while at the same time providing clear and easy to understand information to potential customers.

This calculator uses group size tiers (under 10 to 300+ employees) and an adjustable utilization slider to instantly calculate annual savings estimates. The interface includes responsive design for presentations and smooth visual feedback as users adjust parameters. All calculations happen client-side for immediate results.
